# 📊 HR Analytics Dashboard

## 📌 Project Overview

This project presents an interactive **HR Analytics Dashboard** built using **Power BI** to analyze employee data and provide insights into workforce trends, employee performance, hiring patterns, promotions, and attrition.

The dashboard helps HR managers and business stakeholders make data-driven decisions by identifying key workforce metrics and areas that require attention.

---

## 🎯 Project Objectives

- Monitor employee attrition across departments.
- Track hiring trends over time.
- Evaluate employee performance.
- Measure promotion and training effectiveness.
- Build an interactive dashboard for business decision-making.

---

## 🛠️ Tools & Technologies

- **Power BI**
- **Power Query**
- **DAX**
- **Microsoft Excel / CSV**
- **Git & GitHub**

---

## 📂 Dataset Information

The dataset contains employee-related information, including:

- Employee ID
- Department
- Job Role
- Gender
- Salary
- Hire Date
- Performance Score
- Training Hours
- Promotion Status
- Attrition Status

---

## 📈 Dashboard KPIs

The dashboard includes the following KPI cards:

- 👥 Total Employees
- 📉 Attrition Rate
- 💰 Average Salary
- ⭐ Average Performance Score

---

## 📊 Dashboard Visualizations

### 1. Attrition by Department
Displays the number of employees who left the organization across departments.

### 2. Hiring Trend
Shows yearly/monthly employee hiring trends.

### 3. Average Salary by Job Role
Compares average salaries across different job roles.

### 4. Performance Score Distribution
Displays employee performance ratings.

### 5. Training Hours vs Performance
Analyzes the relationship between employee training and performance.

### 6. Promotion Status
Shows the percentage of promoted and non-promoted employees.

---

## 🎛 Interactive Filters (Slicers)

The dashboard allows users to filter data using:

- Department
- Job Role
- Gender
